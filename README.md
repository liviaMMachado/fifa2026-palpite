## Frontend + API

O projeto inclui um frontend web integrado ao backend Java existente.

Para executar clique em run-server.bat na pasta do projeto e abra o servidor em http://localhost:8080.

## Terminal Manual

Se javac não for reconhecido, use o Java do IntelliJ:
execute o Main.java no terminal.

## Fluxo de telas

1. Iniciar — tela inicial
2. Jogos do dia — escolha a partida (dados vindos da API)
3. Seu palpite — ajuste o placar 00 x 00 e clique em Enviar
4. Enviado com sucesso — exibido por 10 segundos
5. Fazer outro palpite — retorna à lista de jogos
